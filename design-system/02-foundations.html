Fundamentals · 02

Visual foundations

Three surfaces share primitives (spacing, radii, motion) but keep their own color and type. This is intentional — the admin panel should feel like a precision tool; the marketing site should feel like a magazine.

2.1 Color

Three palettes, three jobs.

A single global palette was the wrong answer — the marketing site wants warmth and white space, the admin wants GitHub's quiet dark, and each photographer theme has its own mood. We tokenize per surface so tokens never leak.

2.2 Type

Inter for work, Playfair for story.

Across Marketing and Admin we use one sans (Inter) paired with one serif (Playfair Display). The serif is reserved for headlines and brand marks — using it anywhere else dilutes it. Photographer themes each have their own pairings, documented below.

The italic move

Playfair italic is the brand signature. It appears exactly once per long-form page — inside a display headline, usually as the emotional tail: "Tudo em 1 só lugar.". Never italicize body copy for emphasis; use weight instead.

2.3 Spacing

4-pixel base. Ten stops. Stop inventing numbers.

Every gap, every padding, every margin resolves to a token. If a designer hand-types 18px, they're working around the system.

Tight UI (admin)
Use sp-1 to sp-4. Nav items: sp-2 vertical. Card interior: sp-4.
Marketing sections
Section padding: sp-8 vertical. Card interior: sp-5. Hero padding: sp-9.
Photographer sites
Section padding per theme; Elegante = sp-8, Escuro = sp-9, Minimalista = sp-7.

2.5 Shadow

Three shadows, no more.

Shadows indicate elevation only. A drop shadow on a card means this can move. If nothing can move, no shadow.

shadow-smhairline lift · cards at rest
shadow-mdhover, dropdown menus
shadow-lgmodals, floating nav, pricing card hover
Admin exception: on dark (#0d1117), shadows invert to 0 24px 80px rgba(0,0,0,0.8) — only the preview iframe and login card use this.
2.6 Motion

Fast. Out. Done.

--dur-fast · 150ms
Hover states, button press feedback, small transitions.
--dur-base · 250ms
Card lift, dropdown open, tab switch.
--dur-slow · 400ms
Modal open, sidebar collapse, preview pane morph.
--ease-out
cubic-bezier(0.22, 1, 0.36, 1) — default for things arriving.
--ease-in-out
cubic-bezier(0.65, 0, 0.35, 1) — for things both appearing and leaving.
Reduced motion: always honor prefers-reduced-motion; disable all transforms and reduce durations to 10ms.
